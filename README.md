Инструкция:
1. Длеаем свой форк при помощи кнопки в правом верхнем углу со словом fork
2. Клонируем репозиторий
3. Создаём папку, называем своей фамилией и в неё переносим код из папки sample
4. В файлах .h и .cpp вставляем свой код, названия методов НЕ МЕНЯЕМ
5. Когда всё сделали - делаем commit, push и pull request
